gaussian_kernel_norm ( n,
m,
sigma )

Generates a normalized 2D Gaussian kernel centered at the pixel (n, m) with standard deviation sigma.

This function creates a Gaussian kernel that is centered at the specified pixel coordinates (n, m). The kernel is generated based on the standard deviation (sigma) and is normalized to ensure that the sum of all values in the kernel equals 1.

Parameters
nHeight of the kernel (number of rows).
mWidth of the kernel (number of columns).
sigmaStandard deviation of the Gaussian function.
Returns
: Normalized 2D Gaussian kernel.
