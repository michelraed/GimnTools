rot_center ( theta)

Calculates the center of rotation (COR) of a sinogram.

This function computes the center of rotation from a 2D sinogram array by analyzing its Fourier transform.

Parameters
thetasum(array-like) The 2D sinogram array with dimensions (z, theta).
Returns
float The calculated center of rotation (COR).

The function applies a real-valued Fast Fourier Transform (rFFT) to the flattened input array, extracts the real and imaginary parts at a specific frequency, computes the phase, and then determines the center of rotation based on the phase information.
