apply_filter ( kernel,
image )

Implements a convolutional filter that applies a kernel to an image.

This function applies the specified kernel to the input image using convolution. The kernel must have the same dimensions in both x and y.

Parameters
kernelThe convolution kernel to apply.
imageThe input image to which the kernel will be applied.
Returns
: The filtered image.

v,
window )

Extracts the neighboring pixels from an image centered at pixel (u, v) within a specified window size.

This function creates a matrix of neighbors from the image, centered at the pixel (u, v), with a size defined by the window parameter.

Parameters
imagemInput image from which to extract neighbors.
uRow index of the center pixel.
vColumn index of the center pixel.
windowSize of the window (should be odd).
Returns
: A matrix containing the neighboring pixels.
