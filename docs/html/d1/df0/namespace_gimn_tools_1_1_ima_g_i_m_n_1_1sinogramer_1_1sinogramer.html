convert_crystal_to_xy ( crystal_id)

Converts a crystal ID into (x, y) coordinates on a grid.

This function calculates the (x, y) coordinates corresponding to a given crystal ID on an 8x8 grid. The crystal ID is used to determine the position on the grid, where x represents the column and y represents the row.

Parameters
crystal_idThe ID of the crystal to be converted. It should be a non-negative integer.
Returns
A NumPy array containing the (x, y) coordinates of the crystal.
  • out[0]: x-coordinate (column)
  • out[1]: y-coordinate (row)
Note
The function assumes that crystal IDs range from 0 to 63, corresponding to an 8x8 grid (a total of 64 positions).
